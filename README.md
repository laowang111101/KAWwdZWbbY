# Python人脸识别系统 python407

本项目是基于Python的人脸识别系统，支持摄像头实时识别和上传图片识别功能。用户可注册人脸信息至数据库进行比对，系统功能齐全，操作界面友好。

## 技术栈

- 编程语言：Python
- 人脸识别库：OpenCV、Face Recognition

## 功能模块

- 人脸注册
- 实时识别
- 上传图片识别
- 日志导出

## 系统角色

- 用户：进行人脸注册、识别操作
- 管理员：管理用户数据和系统日志

## 运行环境

- 操作系统：Windows、Linux、macOS
- Python版本：3.6及以上
